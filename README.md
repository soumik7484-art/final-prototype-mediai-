# рџ§  About MediAI

MediAI is an intelligent healthcare assistant designed to simplify medical understanding using Artificial Intelligence.

The platform combines OCR, Computer Vision, and Large Language Models to help users understand prescriptions, analyze medical images, and receive reliable health guidance through an interactive AI chatbot.

Whether you're a patient, student, or healthcare professional, MediAI makes medical information easier to access and understand.
